# Logos dos clientes

Coloque aqui os logos dos clientes. Use **PNG com fundo transparente** sempre que possível
(fica muito melhor sobre o fundo escuro do site). Tamanho recomendado: altura de ~200px.

Salve cada arquivo com **exatamente** estes nomes (tudo minúsculo):

| Cliente | Nome do arquivo |
|---------|-----------------|
| Venezza Pizzaria | `venezza.png` |
| Conceito Beer | `conceito-beer.png` |
| Chicago Pizzaria | `chicago.png` |
| Meu Canto Casa do Espeto | `meu-canto.png` |
| Beach Pet Store | `beach-pet.png` |
| World Car | `world-car.png` |
| Victorine | `victorine.png` |
| Villa Maria Pub | `villa-maria.png` |
| Delany Lingerie | `delany.png` |
| Le Penariol | `le-penariol.png` |

Enquanto o arquivo não existir, o site mostra automaticamente o **nome do cliente em texto**
(não quebra nada). Assim que você subir o arquivo com o nome certo, o logo aparece sozinho.
